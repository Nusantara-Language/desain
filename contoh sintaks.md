## Contoh Sintaks
```nusantara
bilangan umur. // Membuat variabel tanpa mengisi nya.
teks nama = 'Fern Aerell'. // Membuat variabel dengan isi.
umur = 15. // Mengisi variabel umur.

// Membuat fungsi
halodunia() cetak('Halo Dunia!').
tambah(bilangan x, y) => x + y.
kurang(bilangan x, y) => x - y.
bagi(bilangan x, y) {
    => x / y.
}.

// Memanggil fungsi
halodunia().

// Membuat kelas
kelas Manusia {
    bilangan darah.
    Manusia(bilangan darah) cakupan.darah = darah.
    fungsi berjalan cetak "Manusia Berjalan...".
}
```
